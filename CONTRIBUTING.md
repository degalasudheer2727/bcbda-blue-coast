# Contributing to BCBDA

Thank you for helping improve the Bapatla–Chirala Beach Development Authority planning concept. This is an **open planning artifact**, and contributions that improve accuracy, add evidence, or strengthen the plan are welcome.

## Ways to contribute

- **Correct or source a fact** — open an issue or PR with a citation. We prefer primary/government sources.
- **Add or refine data** — improve the JSON datasets in `/data`.
- **Strengthen the plan** — propose edits to documents in `/docs` with reasoning.
- **Improve the website** — fix bugs or enhance `/website` (Next.js).

## Ground rules

1. **Cite sources** for factual claims. Mark planning assumptions clearly as "indicative."
2. **Respect ecology & community.** Proposals must keep the "ecology first" and "community ownership" principles intact.
3. **No false attribution.** BCBDA is a *proposed* body; do not present this material as an official Government of Andhra Pradesh decision.
4. **Be constructive and respectful** (see `CODE_OF_CONDUCT.md`).

## Workflow

1. Fork → branch → commit with a clear message.
2. For docs, keep the existing structure and tone.
3. For the website, run `npm run build` in `/website` before submitting.
4. Open a pull request describing *what* changed and *why*.
