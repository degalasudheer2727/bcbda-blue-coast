# Code of Conduct

## Our pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone, regardless of background, identity, or experience level.

## Our standards

Examples of behaviour that contributes to a positive environment:
- Using welcoming and inclusive language.
- Respecting differing viewpoints, especially from coastal communities and domain experts.
- Accepting constructive criticism gracefully.
- Focusing on what is best for the coast, the community, and the public interest.

Unacceptable behaviour includes harassment, personal attacks, dismissing community or ecological concerns, and publishing others' private information.

## Scope

This Code applies within all project spaces (issues, pull requests, discussions) and when representing the project publicly.

## Enforcement

Maintainers are responsible for clarifying standards and may remove, edit, or reject contributions that violate this Code. Report concerns by opening a confidential issue or contacting the maintainers.

This Code of Conduct is adapted from the Contributor Covenant, version 2.1.
